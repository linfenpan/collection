CSS3配套动画

属性配置

<div data-ctr="enter x|20px">da宗熊</div> 一句属性，进入、离开两种动画

多种动画

<div data-ctr="enter x|-50px s|1.1 r|40deg">..</div> 支持透明、translate、scale、rotate 4种动画

多设备支持

支持IE10及以上版本IE
支持Firefox、Opera 和 Chrome
移动多友爱
对支持transition的浏览器友好~

可控性

var st = new Css3Transition();
st.enter(); // 进入 st.leave(); // 离开
